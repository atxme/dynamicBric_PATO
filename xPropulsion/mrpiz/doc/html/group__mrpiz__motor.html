◆ mrpiz_motor_encoder_get()

int mrpiz_motor_encoder_get ( mrpiz_motor_id  id)

Donne la valeur d'un codeur de moteur.

La valeur d'un codeur se situe dans l'intervalle [-2147483648, 2147483647]. La gestion d'erreur se fait avec errno (cf. error.h).

Paramètres
idl'identifiant du moteur concerné

Remarque : L'identifiant MRPIZ_MOTOR_BOTH n'est pas considéré comme un identifiant valide pour cette fonction (errno = MRPIZ_E_MOTOR_ID).

Renvoie
la valeur du codeur, errno est renseigné en cas d'erreur (cf. error.h)

◆ mrpiz_motor_set()

int mrpiz_motor_set ( mrpiz_motor_id  id,
int  cmd 
)

Envoi une consigne à un moteur pour qu'il tourne à une certaine vitesse dans un certains sens.

Fixe la consigne de rotation d'un moteur en pourcentage :

  • une consigne négative fait tourner le moteur vers l'arrière ;
  • une consigne positive fait tourner le moteur vers l'avant ;
  • une consigne nulle arrête le moteur.

Remarque : L'identifiant MRPIZ_MOTOR_BOTH permet de commander les deux moteurs.

Paramètres
idl'identifiant du moteur
cmdla consigne en pourcentage à appliquer au moteur
Renvoie
0 en cas de success, -1 en cas d'erreur (cf. error.h)
